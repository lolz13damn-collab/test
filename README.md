# Telegram бот для получения текущей погоды
## Описание
Бот написан с использованием библиотеки [telebot](https://github.com/python-telegram-bot/python-telegram-bot)

## Технологии
Использовались данные из Api openweather.org

## Как запустить
В файле key.py, подставляете свой токен и api ключ
Чтобы получить токен, нужно написать боту @BotFather в телеграмме и следовать дальнейшим инструкциям
Чтобы получить ключ от api нужно перейти на https://openweathermap.org/api и зарегистрироваться там
